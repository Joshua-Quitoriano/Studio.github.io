## Introduction <a name="introduction"></a>

The Graduation Pictorial Management System is a comprehensive platform for managing graduation photo sessions, appointments, and payments. This guide provides detailed instructions for both students and administrators.

### System Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Stable internet connection
- Supported image formats: JPG, PNG (max 5MB)

## Student Guide <a name="student-guide"></a>

### 1. Account Management
- **Registration**: 
  - Click "Register" on the login page
  - Fill in required details (name, email, student ID)
  - Upload a valid school ID for verification
  - Wait for admin approval

- **Login**:
  - Use your registered email/username and password
  - Reset password option available via email

### 2. Scheduling Appointments
1. Navigate to "Book Appointment" from dashboard
2. Select preferred date and time
3. Choose photo package
4. Submit appointment request
5. Track status in "My Appointments"

### 3. Payment Process
1. After appointment approval, upload payment receipt
2. Supported payment methods:
   - Bank transfer
   - Online payment
   - Over-the-counter
3. Wait for payment verification
4. Download payment confirmation

### 4. Photo Gallery
- View approved photos in "My Gallery"
- Download photos in various sizes
- Share photos via secure link
- Request photo retouching

## Admin Guide <a name="admin-guide"></a>

### 1. User Management
- View all registered users
- Approve/reject registrations
- Manage user roles
- Reset user passwords
- Deactivate accounts

### 2. Appointment Management
- View appointment calendar
- Approve/reject appointments
- Manage scheduling conflicts
- Send notifications
- Generate daily schedule

### 3. Payment Verification
1. Access payment dashboard
2. View uploaded receipts
3. Verify payment details
4. Update payment status
5. Generate payment reports

### 4. Photo Management
- Upload photos to student galleries
- Manage photo packages
- Process retouching requests
- Archive completed sessions

### 5. Reporting
- Generate various reports:
  - Daily appointments
  - Payment statistics
  - Student completion rates
  - Revenue reports
- Export to PDF/Excel
- Schedule automated reports

## Security Best Practices <a name="security"></a>

### For Students
1. Use strong passwords
2. Don't share login credentials
3. Log out after each session
4. Report suspicious activities
5. Keep payment receipts secure

### For Admins
1. Regular password updates
2. Two-factor authentication
3. Monitor system logs
4. Regular backup checks
5. Verify user identities

## Troubleshooting <a name="troubleshooting"></a>

### Common Issues

1. **Cannot Login**
   - Check caps lock
   - Clear browser cache
   - Reset password if needed
   - Contact admin if persistent

2. **Upload Errors**
   - Check file size (max 5MB)
   - Verify file format
   - Try different browser
   - Check internet connection

3. **Payment Issues**
   - Verify payment amount
   - Check receipt format
   - Contact admin for assistance
